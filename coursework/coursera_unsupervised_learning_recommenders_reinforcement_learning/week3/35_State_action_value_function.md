## Title
State-action value function

## Description
- Q(s, a)
- = the return if you
	- start in state s
	- take action a (once)
	- behave optimally after that

Maximize Q(s, a)

pi(s) = a

## Additional Notes
Requires the ability to calculate Q(s,a) for every state and action

sometimes called Q*, or optimal Q function
